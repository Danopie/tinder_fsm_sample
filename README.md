# tinder_fsm_sample

A project to implement Tinder approach to state management.

## Flight Product Flow

* Select flight
    * Loading flights
    * Idle
    * Completed
* Select passenger count
    * Loading passenger threshold
    * Idle
    * Completed
* Enter contact info
    * Idle
    * Validating
    * Completed
* Process Payment
    * Load payment data
    * Verify payment data
    * Pending for confirm
    * Charge purchase
    * Verify receipt
    * Completed
* Completed
